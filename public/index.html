🚀 all_subs

VPN-подписка для белых списков в РФ (через Vercel)

ℹ️ О проекте: Набор серверных функций Vercel для доступа к подпискам all_subs. Все эндпоинты загружают данные напрямую с GitHub при каждом запросе.

📦 Доступные эндпоинты

1. Полный список (обычный текст)

https://solovyov-jenya2004.vercel.app/final_sorted

2. Полный список в Base64

https://solovyov-jenya2004.vercel.app/final_sorted_base64

3. Мобильная подписка (случайная выборка)

Каждый раз отдаёт новый случайный набор конфигов из общей базы. По умолчанию — 100 конфигов.

https://solovyov-jenya2004.vercel.app/random

С параметром количества:

https://solovyov-jenya2004.vercel.app/random?n=50
https://solovyov-jenya2004.vercel.app/random?n=200

🧠 Как это работает

  • /final_sorted – загружает полный список конфигов и отдаёт его как обычный текст
  • /final_sorted_base64 – делает то же самое, но кодирует результат в Base64
  • /random – загружает полный список, перемешивает его и возвращает случайную выборку (параметр ?n=)
🔄 Обновления: Основной генератор подписок работает в all_subs и запускается каждую минуту.
